# Digital CRRSA

Civil registration platform branding and frontend.

## Backend API

A complete Express + PostgreSQL backend is available in `backend`.

Quick setup:

1. Copy `backend/.env.example` to `backend/.env`
2. Run `npm run backend:install`
3. Configure PostgreSQL and run `npm run backend:db:init`
4. Start backend with `npm run dev:backend`
